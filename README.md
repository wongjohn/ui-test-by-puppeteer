# ui-test-by-puppeteer

使用Puppeteer进行前端界面测试。

## 安装、执行方法

```bash
# 建议使用cnpm进行安装，puppeteer需要下载Chromium (~170Mb Mac, ~282Mb Linux, ~280Mb Win)
sudo npm install -g cnpm

cnpm install

npm start
# or npm run test
```
